## Featured Projects

### 1. [Clinic Room Scheduling Automation](https://github.com/miangelisse-ux/clinic-room-scheduling-automation)
**Description:** Automates Google Calendar room scheduling for a clinical environment, preventing double-booking and sending daily summary emails to staff.  
**Tech Stack:** Google Apps Script, Google Calendar API, Google Forms triggers  
**Highlights:**
- Multi-room fallback logic
- Buffer time enforcement
- Conflict detection and resolution
- Automated emails to staff and admin alerts

---

### 2. [Annual Rollup Automation](https://github.com/miangelisse-ux/annual-rollup)
**Description:** Aggregates financial data from multiple Excel workbooks to generate insurance, patient, and monthly summary reports.  
**Tech Stack:** PowerShell, Microsoft Excel COM Interop  
**Highlights:**
- Reads multiple Excel files and normalizes headers
- Aggregates insurance and MRN balances
- Calculates monthly posted amounts
- Outputs formatted, ready-to-use Excel reports
